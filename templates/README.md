# Templates

Copy-paste skeletons for common Flow ID Reveal artifacts.

| File | Purpose |
|---|---|
| [org-pair-map.json](org-pair-map.json) | Skeleton for an org-pair Id mapping. Import via Patch → Org-pair maps → Import from clipboard. |
| [flow-backup.json](flow-backup.json) | Skeleton showing the shape of an Export → Backup output. |
| [package.xml](package.xml) | MDAPI package manifest for a single-flow deploy. |
| [cmt-environment-bindings.csv](cmt-environment-bindings.csv) | Sample data for the Auto-pin-to-CMT pattern (Custom Metadata Type records). |
| [deploy-runbook.md](deploy-runbook.md) | Pre-deploy / deploy / post-deploy checklist template. |
| [draft-naming-convention.md](draft-naming-convention.md) | Recommended Draft label + description template for staging Drafts. |

Each template includes inline comments. Edit for your situation.
