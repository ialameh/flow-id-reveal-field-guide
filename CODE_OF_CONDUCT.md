# Code of Conduct

This project follows the spirit of the Contributor Covenant. The short version:

## Our pledge

We commit to making participation in this project a respectful experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity.

## What we expect

- **Be kind.** Disagree about content, not about people.
- **Stay on topic.** This project is about Agentforce documentation. Keep discussions relevant.
- **Be patient.** People come to this guide at different levels of experience. Answer questions as if the asker is competent but new to the topic.
- **Give credit.** When someone contributes an idea, fix, or example, name them in commit messages or release notes.

## What is not okay

- Harassment, discrimination, or personal attacks.
- Trolling, deliberate provocation, or bad-faith argument.
- Sharing private information without permission.
- Sustained disruption of discussions.

## How issues are handled

If someone violates the code of conduct, the maintainers will:

1. Privately ask the person to stop the behaviour and explain why.
2. If the behaviour continues, issue a warning publicly.
3. If still continuing, ban the person from the project.

Maintainers will use their judgement. The goal is to keep the project useful and welcoming, not to enforce rigid rules.

## Reporting

If you see behaviour that violates this code of conduct, contact the maintainers privately. Do not name the person publicly until the maintainers have had a chance to investigate.

## Maintainers

Maintainers are also bound by this code. If a maintainer violates it, contact another maintainer.
